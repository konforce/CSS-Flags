CSS Country Flag Demo

The flags are all part of a single image:

The names follow the standard ISO 3166-1 alpha-2 codes. Include the CSS, add the proper class names to any element, and you'll get the flag.

e.g., <span class="us flag"></span>

  • Canada
  • Mexico
  • USA

It will work in any modern browser. If you need to work with IE 7 or worse, you may need to adjust the CSS to use floats and blocks—or some other CSS sprite sheet technique that is known to work.
